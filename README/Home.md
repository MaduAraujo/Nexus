# Nexus — Home

Página inicial pública do sistema. Apresenta a plataforma estatísticas, stack tecnológica, funcionalidades, conformidade LGPD e sobre o projeto — com design dark, animações de scroll e navegação responsiva.

---

## Visão Geral

A landing page do Nexus é uma página estática de apresentação voltada ao público externo. Seu objetivo é comunicar o valor da plataforma de RH, exibir as funcionalidades disponíveis e direcionar o usuário para o login. Todo o conteúdo é carregado sem dependências externas de framework — apenas HTML, CSS e JavaScript.
