# Nexus — Tela de Arquivos

Tela de gerenciamento de documentos. Permite upload, visualização, busca e exclusão de arquivos organizados por categoria admissional e demissional.

---

## Visão Geral

A tela de Arquivos é composta por três áreas principais:

- **Cards de estatísticas** — exibem o total de arquivos, admissionais e demissionais
- **Toolbar** — campo de busca
- **Tabela de arquivos** — lista os documentos com ações de visualização e exclusão

---

## Estrutura de Arquivos

```
nexus/

├── styles/

└── javascript/
```
